# Student Analytics Dashboard

A web-based Student Analytics Dashboard developed using Streamlit, Pandas, NumPy, and Matplotlib. The application allows users to manage student records, analyze performance, and visualize data through interactive charts.

## Project Overview

The Student Analytics Dashboard helps users:

- Add and manage student records
- Analyze student performance
- Generate statistical summaries
- Visualize data using charts and graphs
- Explore gender distribution and rankings

## Features

- Student Data Management
- Average Marks Calculation
- Highest and Lowest Marks Analysis
- Student Ranking
- Bar Chart Visualization
- Line Chart Visualization
- Pie Chart Visualization
- Histogram Visualization
- Scatter Plot Visualization
- Download Dataset Option

## Installation

### Clone the Repository

```bash
git clone <repository-url>
cd Student-Analytics-Dashboard
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

Or install manually:

```bash
pip install streamlit pandas numpy matplotlib
```

## Running the Application

Start the Streamlit server:

```bash
streamlit run app.py
```

The application will be available at:

```text
http://localhost:8501
```
